# Wise Mind web page

#### Used technologies

Webpack, ESLint, Prettier
Javascript, ES6,7,8
React, hooks, Redux

Jest, Enzhyme
StoryBook
Node, Express, Mongo, Mangoose (separate repo)

# Install guide
```sh
$ git clone ...
$ npm install
$ npm start
```
